Defina diferentes taxas de juros de acordo com o número de parcelas

== Description ==

Com este plugin você consegue definir taxas personalizadas em cada mês de parcelamento do seu site. Em alguns gateways isso é necessário.

Este plugin requer a instalação do plugin [WC Simulador de Parcelas](https://fernandoacosta.net/produto/simulador-parcelas/).

O simulador de parcelas para WooCommerce exibe as opções de pagamento ao cliente tanto para parcelamento quanto descontos à vista. No entanto, a aplicação dessas regras no carrinho e checkout são responsabilidade dos gateways de pagamento instalados.

== Installation ==

e.g.

1. Upload \`plugin-name.php\` to the \`/wp-content/plugins/\` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place \`<?php do_action('plugin_name_hook'); ?>\` in your templates
